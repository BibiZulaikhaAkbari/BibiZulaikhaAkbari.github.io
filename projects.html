4. Hamilton Waterfalls Map Application

Description: Built a map application showcasing waterfalls in Hamilton using geolocation and geocoding APIs.

Programming Language: JavaScript (Geolocation API, Geocoding API), Bootstrap for styling

Features:

  • Displays waterfalls with markers on a map, differentiated by type (cascade or waterfall) using unique images in the info windows.
  • Provides an info window for each marker, showing the name, city, type, and an image of the waterfall type.
  • Includes a "Get Directions" button that navigates the user to their desired waterfall.
  • Geolocates the user for directional purposes and prompts location access if not already enabled.
  • Allows users to search for specific waterfalls and dynamically adds markers to the map based on their requests.
